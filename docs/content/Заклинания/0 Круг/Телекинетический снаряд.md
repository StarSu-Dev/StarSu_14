# Телекинетический снаряд
**Мистик 0 круг**

|  |  |
|----------|----------|
| **Школа** | Разрушение |
| **Время сотворения** | 1 основное действие |
| **Дистанция** | Близкая (25 футов + 5 футов за 1 УЗ) |
| **Цель** | Одно существо и один предмет |
| **Длительность** | Мгновенная |
| **Испытание** | Нет |
| **Устойчивость к магии** | Нет |

## Описание

Вы можете швырнуть в цель бесхозный предмет весом менее 1 (до 5 фунтов или менее), совершив дистанционную атаку против ККБ цели. При попадании цель, и предмет получают 1d6 дробящего урона. Тип предмета никак не влияет ни на саму атаку, ни на её урон.
